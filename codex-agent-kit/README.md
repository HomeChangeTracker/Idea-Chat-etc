# Codex Agent Kit (개인 맞춤형 프로젝트 운영용)

이 폴더는 앞으로 프로젝트를 시작할 때 Codex와 빠르게 협업하기 위한 템플릿 모음입니다.

사용 순서(추천):

1. `01-project-profile-survey.md` 작성
2. `02-quick-start-checklist.md` 체크
3. `03-codex-prompt-templates.md`의 프롬프트로 Codex에 `AGENTS.md` 생성 요청
4. 필요하면 `04-agents-md-template.md` / `05-agents-override-template.md`를 직접 편집
5. 반복 작업이 보이면 `06-skill-ideas-and-scope.md`로 스킬 후보 정리

메모:

- 처음부터 완벽하게 채우지 않아도 됩니다. `미정`으로 남겨도 됩니다.
- 규칙은 추상적인 원칙보다 실행 가능한 기준(명령/조건)으로 적는 것이 좋습니다.
- 프로젝트별 예외는 루트 `AGENTS.md`보다 하위 `AGENTS.override.md`로 분리하는 방식이 유지보수에 유리합니다.
