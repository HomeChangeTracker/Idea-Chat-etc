아래 템플릿은 복붙해서 바로 사용할 수 있게 작성했습니다.

## 1) AGENTS.md 생성 요청 (설문 기반)

```md
내가 아래 규칙으로 프로젝트를 진행하려고 해.
이 내용을 바탕으로 리포지토리 루트용 `AGENTS.md`를 만들어줘.

조건:
- 실행 가능한 규칙(명령/체크 기준) 중심으로 써줘
- 모호한 항목은 먼저 질문 5개 이내로 물어봐
- 내가 `미정`이라고 쓴 항목은 합리적인 기본값을 제안하되, 제안임을 표시해줘
- 최종 출력은 `AGENTS.md` 본문과, 선택적으로 분리하면 좋은 `AGENTS.override.md` 제안까지 포함해줘

[설문 답변 붙여넣기]
```
