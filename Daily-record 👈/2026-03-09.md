## 사전 작업: 백엔드 로컬 구동 트러블슈팅 (박진현 공유)
- `feat/inference` 코드를 로컬에서 구동하며 발생한 에러와 해결책을 팀원들에게 공유
- 필수 파일 누락 문제(`ImportError`)는 이전 PR에서 누락된 `config.py`와 `database.py`를 직접 경로에 덮어씌워 해결
- Pydantic 환경 변수 에러(`ValidationError`)는 누락된 변수를 명시적으로 추가하고, `extra = "ignore"` 속성 설정하여 방어 코드 작성
- 환경 변수 인식 지연 문제는 `main.py` 최상단에 `load_dotenv`를 추가하여 파이썬 실행 시 환경 변수를 강제로 읽도록 조치
- Swagger UI의 파일 업로드 버튼 미노출 버그는 파라미터 타입을 `UploadFile = File(...)` 형태로 수정하여 정상 렌더링되도록 수정

## 프론트엔드 구동 및 연동 이슈
- 연석 님이 프론트 구동 중 겪은 에러는 진현 님의 안내에 따라 `pnpm install` 후 `pnpm dev` 명령어를 실행 후, `.env.local` 파일에 API 주소(`http://localhost:8000/api/v1`)를 설정하여 해결
- 대시보드에서 매물보기 진입 시 사이드 메뉴 사라지는 현상 발생
  - 로그아웃 설정되어 있을 것으로 추정 (구조 수정 진행 예정)
- 우혁 님이 해당 현상의 원인이 컴포넌트 내부의 `isLoggedIn={false}` 설정 때문임을 파악 
- 지금 당장 로그인 로직 구현보다는 모델 성능 지표 확인이 우선이므로, 임시로 값을 `true`로 설정하여 넘어가기로 합의

## 모델링(AI 추론) 구동 및 성능 검토
- 근우 님이 API 호출 시 `CLIP`, `SAM` 등의 모델이 지연 로드(Lazy Load)되도록 세팅되어 있어 최초 실행 시 시간이 소요될 수 있다고 설명
- 연석 님이 모델 다운로드 13초, 추론 2초 만에 모델 결과 추출 성공
- 진현 님도 로컬에서 성공적으로 구동하여 총 4군데의 변화 영역(ROI) 감지, 0.72~0.94 수준의 신뢰도 수치 도출, `artifacts` 내 시각화 이미지 정상 저장 결과 확인
- 모델링 결과, L채널(밝기)을 활용하는 `diff` 추출 과정에서 그림자까지 변화로 잡는 문제 발견
- 또한 정합 시 발생하는 바깥쪽 검은색 여백 부분도 변화로 인식되는 문제가 확인되어, 근우 님이 정렬 알고리즘 튜닝을 다시 시작했고 연석 님도 크롭(Crop) 방식 등을 개별적으로 시도 예정
