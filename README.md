# Idea-Chat-etc
팀프로젝트를 진행하면서 나왔던 의견들 및 기록입니다.

- **Daily-record**
	- 프로젝트 주요사항 및 취업특강 정리 내용이 포함되어 있습니다.
- **Default Link**
	- Google Drive, Figma 등 온라인 링크가 포함되어 있습니다.
	- Obsidian이 내장 브라우저 기능이 포함되어 사용에 용이할 것 같아 포함시켰습니다.
- **추가 기능**
	- 옵시디언 우측 상단을 보면, `calendar` 기능이 있습니다.
	- 해당 기능으로 `일간 기록`이 작성된 날짜를 달력 형태로 쉽게 확인할 수 있습니다.
